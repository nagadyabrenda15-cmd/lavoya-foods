# Lavoya Foods – Web-Based E-Commerce System

A collaborative final-year Bachelor of Business Computing project developed at Makerere University Business School (MUBS).

## Overview

Lavoya Foods is a web-based e-commerce system designed to improve food ordering by allowing customers to browse menus, place orders online, and reduce waiting time at restaurants.

The project was developed to address challenges associated with manual food ordering and improve convenience for customers. :contentReference[oaicite:0]{index=0}
